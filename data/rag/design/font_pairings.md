# What this is

Font pairings are combinations of fonts that work well together for headings and body text in digital products, creating visual hierarchy and professional design.

# When to use

- **Brand kit setup**: Choose 1-2 fonts for your entire product line
- **Document design**: eBooks, workbooks, planners, landing pages
- **Consistency**: Use same fonts across all materials
- **Readability**: Ensure body text is easy to read on screens and print

# 5-step build recipe

1. **Choose heading font**: Bold, distinctive, personality (serif or sans-serif)
2. **Choose body font**: Readable, neutral, professional (usually sans-serif)
3. **Test contrast**: Headings stand out, body is comfortable to read
4. **Apply hierarchy**: H1 (24-36pt), H2 (18-24pt), Body (12-14pt)
5. **Test on devices**: Check readability on mobile, tablet, desktop

# Outline template

```
Heading font: [Bold, attention-grabbing]
Body font: [Readable, neutral]
Hierarchy: H1 > H2 > H3 > Body > Captions
```

# Popular font pairings

**Professional & Clean:**
- Heading: Montserrat Bold + Body: Open Sans
- Heading: Playfair Display + Body: Source Sans Pro
- Heading: Raleway + Body: Lato

**Modern & Minimal:**
- Heading: Inter + Body: Inter (same font, different weights)
- Heading: Poppins + Body: Roboto
- Heading: Space Grotesk + Body: Work Sans

**Elegant & Sophisticated:**
- Heading: Cormorant Garamond + Body: Montserrat
- Heading: Cinzel + Body: Lora
- Heading: Libre Baskerville + Body: Source Sans Pro

**Friendly & Approachable:**
- Heading: Quicksand + Body: Nunito
- Heading: Comfortaa + Body: Open Sans
- Heading: Fredoka + Body: Karla

# Font selection rules

- **Max 2 fonts**: One for headings, one for body (or same font, different weights)
- **High contrast**: Serif + sans-serif OR bold + light weights
- **Google Fonts**: Free, commercial-use, web-safe
- **Readability first**: Avoid overly decorative fonts for body text

# Common mistakes

- Too many fonts (3+ creates chaos)
- Low contrast (headings don't stand out)
- Decorative body text (hard to read long paragraphs)
- No font hierarchy (everything same size)

# Metrics to watch

- Reader feedback (easy to read?)
- Time on page (comfortable reading experience)
- Completion rate (did they finish reading)
- Brand perception (professional, trustworthy?)
