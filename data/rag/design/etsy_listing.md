# What this is

Etsy is a marketplace for selling digital downloads (planners, templates, printables). Optimizing your Etsy listing helps you rank in search and convert browsers into buyers.

# When to use

- **Selling printables**: Planners, calendars, art prints, worksheets
- **High search volume**: Millions of buyers searching for digital products
- **SEO-friendly**: Tags and keywords drive organic traffic
- **Passive income**: List once, sell unlimited copies

3. **Description**: What's included, how to use, file formats

# Etsy SEO tips

- **Title**: Front-load keywords (first 60 characters matter most)
- **Tags**: Use all 13 tags, mix broad and specific keywords
- **Categories**: Choose the most relevant category
- **Attributes**: Fill in all available attributes (color, occasion, etc.)
- **Shipping**: Set to "digital" for instant download

# Common mistakes

- Generic title (no keywords, not descriptive)
- Only 1-2 photos (need 10 to show value)
- No preview (customers want to see inside)
- Wrong category (reduces visibility)
- Not using all 13 tags (missing search opportunities)
